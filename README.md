Steps to install "Recall Registry Formatter"
============================================

Pre-requiste
------------

	NodeJS 10 + version
	Ptython 2.7.x
	Python dev tools : pip

> Python packages:

	1. pip install csv
	2. pip install re
	3. pip install xldr

> Node Pacakges:

	1. npm install http
	2. npm install formidable
	3. npm install shelljs

Files inside the package
-----------------------

	1. server.js
	2. recall.py
	3. forms.html
	4. package.json
	5. recallreq.txt
